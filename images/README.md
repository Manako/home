# Images Folder

Place your images in this folder with the following names:

## Production Work Section
- `production-1.jpg` (or .png) - Image for first production project
- `production-2.jpg` (or .png) - Image for second production project
- `production-3.jpg` (or .png) - Image for third production project

## Game Design Section
- `design-1.jpg` (or .png) - First game design project image
- `design-2.jpg` (or .png) - Second game design project image
- `design-3.jpg` (or .png) - Third game design project image

## Blog Section
- `blog-1.jpg` (or .png) - Featured image for first blog post
- `blog-2.jpg` (or .png) - Featured image for second blog post
- `blog-3.jpg` (or .png) - Featured image for third blog post

## About Me Section
- `profile.jpg` (or .png) - Your profile photo

## Image Recommendations
- **Production/Blog images**: Recommended size 1200x800px or similar (16:9 aspect ratio)
- **Game Design images**: Recommended size 800x600px or similar (4:3 aspect ratio)
- **Profile photo**: Recommended size 800x800px or similar (square)

You can use .jpg, .png, .webp, or any other web-compatible image format. Just make sure to update the file extensions in `index.html` if you use a different format.
